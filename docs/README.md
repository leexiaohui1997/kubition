## 🎮 原游戏简介

《超苦逼冒险者》是一款基于HTML5的放置类角色扮演游戏：
- **300+** 种物品（装备、材料、食物）
- **16层** 地牢探险
- **25+** 可探索地点
- **完整制造系统**（铁匠、炼金、魔法、科技）

## 🔥 复刻项目目标

| 项目 | 说明 |
|------|------|
| **技术栈** | Vite + React + TypeScript + Zustand + Tailwind CSS |
| **交互形式** | 纯文字MUD风格 + ASCII艺术 |
| **开发方式** | 渐进式迭代，每个里程碑都有可玩版本 |
| **预计工期** | 20-25天获得完整可玩版本 |

## 📊 里程碑速览

| 阶段 | 名称 | 工期 | 主要交付物 |
|------|------|------|------------|
