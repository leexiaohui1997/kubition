# 《超苦逼冒险者》复刻项目 - 总体规划

## 🎯 项目愿景

复刻《超苦逼冒险者》为一款**纯文字风格**的放置类RPG，保留核心玩法循环，采用现代前端技术栈重新实现。

### 核心体验
> 你在一个陌生的世界醒来，四周是安静的森林。你需要采集资源、制作工具、探索未知，在地牢的深处寻找传说中的宝物...

---

## 🛠️ 技术选型

| 类别 | 技术 | 选择理由 |
|------|------|----------|
| **构建工具** | Vite 5+ | 极速开发体验，原生ESM |
| **前端框架** | React 18 + TypeScript | 类型安全，组件化开发 |
| **状态管理** | Zustand | 轻量简洁，TypeScript友好 |
| **样式方案** | Tailwind CSS | 快速构建UI，文字风格易实现 |
| **动画效果** | Framer Motion | 文字滚动、过渡动画 |
| **本地存储** | localStorage | 游戏存档持久化 |
| **字体** | 等宽字体 (JetBrains Mono) | 纯文字MUD风格 |

---

## 📱 交互设计理念

### 纯文字MUD风格

```
┌─────────────────────────────────────────────┐
│  超苦逼冒险者                    [生命: 100/100]  │
├─────────────────────────────────────────────┤
│                                           │
│  你站在银溪镇的中心广场。                  │
│  北风呼啸，街上行人寥寥。                   │
│                                           │
│  你可以：                                    │
│  > [1] 前往静谧森林采集木材                │
│  > [2] 前往铁匠铺制作装备                   │
│  > [3] 与村长交谈                            │
│  > [4] 查看背包                             │
│                                           │
│  最近日志：                                  │
│  [10:23] 你获得了 5 木材                    │
│  [10:21] 体力已恢复                          │
│                                           │
└─────────────────────────────────────────────┘
```

### 设计原则
1. **极简主义** - 仅用文字和简单符号
2. **沉浸感** - 详细的文字描述，氛围营造
3. **响应式** - 支持键盘操作和点击
4. **即时反馈** - 每个操作都有文字反馈

## 🎨 视觉风格指南

### 配色方案（暗色主题）
```css
/* 背景 */
bg-primary: #1a1a2e;    /* 深蓝黑 */
bg-secondary: #16213e;  /* 次背景 */
bg-tertiary: #0f3460;   /* 高亮背景 */

/* 文字 */
text-primary: #e94560;  /* 主强调色（红色） */
text-normal: #ecf0f1;   /* 正常文字 */
text-dim: #95a5a6;      /* 次要文字 */

/* 状态 */
success: #27ae60;
warning: #f39c12;
danger: #e74c3c;
info: #3498db;
```

### 字体层级
```css
/* 标题 */
font-size: 1.5rem;
font-weight: bold;

/* 正文描述 */
font-size: 1rem;
line-height: 1.6;

/* 选择项 */
font-size: 1rem;
color: #3498db;  /* 蓝色链接感 */

/* 日志/状态 */
font-size: 0.875rem;
color: #7f8c8d;
```

## ⚙️ 架构设计预览

```
src/
├── main.tsx                 # App入口
├── App.tsx                  # 主容器（整体布局骨架）
├── index.css                # 全局样式（mud-panel/mud-btn/mud-log 等复用类）
│
├── stores/                  # Zustand 状态管理
│   └── gameStore.ts         # 游戏主状态（玩家/背包/装备/日志/时间）
│
├── constants/               # 集中管理的常量
│   ├── game.ts              # 游戏数值常量（状态上限、消耗速率、进度条参数等）
│   ├── labels.ts            # 中文标签映射（枚举 key → 中文显示名）
│   └── styles.ts            # UI 样式配置（颜色映射、状态栏配置等）
│
├── types/                   # TypeScript 类型定义
│   ├── index.ts             # 类型导出入口
│   ├── game.ts              # 游戏数据类型（Item, Monster, Place, Recipe 等）
│   └── player.ts            # 玩家相关类型（PlayerState, StatConfig 等）
│
├── components/              # React 组件
│   ├── StatusBar.tsx         # 状态栏（四项属性进度条 + 游戏时间 + 警告）
│   ├── LogPanel.tsx          # 日志面板（BetterScroll 滚动）
│   ├── Inventory.tsx         # 背包面板（物品列表 + 使用/丢弃）
│   ├── ItemTooltip.tsx       # 物品信息浮窗（手动定位 + Portal）
│   ├── Modal.tsx             # 通用弹层组件（遮罩 + 动画）
│   ├── TownScene.tsx         # 城镇场景（主场景操作入口）
│   ├── layout/               # 布局组件（预留）
│   ├── scenes/               # 场景组件
│   │   ├── SceneRouter.tsx    # 场景路由（动态切换当前场景）
│   │   ├── TownScene.tsx      # 城镇场景（移动 + 操作）
│   │   ├── ForestScene.tsx    # 森林场景（采集资源）
│   │   └── GenericScene.tsx   # 通用场景（溪流/矿洞等）
│   └── ui/                   # UI 组件（预留）
│
├── data/                    # 游戏数据（全量迁移自原作）
│   ├── items.ts             # 物品定义（409个）
│   ├── monsters.ts          # 怪物定义（99个）
│   ├── places.ts            # 地点定义（20个）
│   ├── recipes.ts           # 制造配方（179个）
│   └── dungeons.ts          # 地牢数据（16层）
│
├── systems/                 # 游戏系统逻辑
│   └── GatheringSystem.ts   # 采集系统（前置检查 + 掉落计算 + 使用物品）
│
├── hooks/                   # 自定义 Hooks
│   ├── useBetterScroll.ts   # BetterScroll 滚动封装
│   ├── usePlayerStatus.ts   # 玩家状态检查与警告
│   ├── useGathering.ts      # 采集进度条逻辑
│   └── useTravel.ts         # 移动进度条逻辑
│
└── utils/                   # 工具函数
    └── dataLoader.ts        # 数据加载与交叉引用验证
```

## 🎮 交互命令设计

### 通用命令
| 命令 | 快捷键 | 说明 |
|------|--------|------|
| 1-9 | 数字键 | 选择对应选项 |
| I | i | 打开背包 |
| C | c | 角色状态 |
| S | s | 系统设置 |
| ESC | esc | 返回/取消 |

### 场景特定命令
**森林场景:**
- [1] 采集木材
- [2] 狩猎兔子
- [3] 采集浆果

**战斗场景:**
- [1] 攻击
- [2] 使用物品
- [3] 逃跑
