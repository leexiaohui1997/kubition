# 《超苦逼冒险者》复刻项目 - 总体规划

## 🎯 项目愿景

复刻《超苦逼冒险者》为一款**纯文字风格**的放置类RPG，保留核心玩法循环，采用现代前端技术栈重新实现。

### 核心体验
> 你在一个陌生的世界醒来，四周是安静的森林。你需要采集资源、制作工具、探索未知，在地牢的深处寻找传说中的宝物...

---

## 🛠️ 技术选型

| 类别 | 技术 | 选择理由 |
|------|------|----------|
| **构建工具** | Vite 5+ | 极速开发体验，原生ESM |
| **前端框架** | React 18 + TypeScript | 类型安全，组件化开发 |
| **状态管理** | Zustand | 轻量简洁，TypeScript友好 |
| **样式方案** | Tailwind CSS | 快速构建UI，文字风格易实现 |
| **动画效果** | Framer Motion | 文字滚动、过渡动画 |
| **本地存储** | localStorage | 游戏存档持久化 |
| **字体** | 等宽字体 (JetBrains Mono) | 纯文字MUD风格 |

---

## 📱 交互设计理念

### 纯文字MUD风格

```
┌─────────────────────────────────────────────┐
│  超苦逼冒险者                    [生命: 100/100]  │
├─────────────────────────────────────────────┤
│                                           │
│  你站在银溪镇的中心广场。                  │
│  北风呼啸，街上行人寥寥。                   │
│                                           │
│  你可以：                                    │
│  > [1] 前往静谧森林采集木材                │
│  > [2] 前往铁匠铺制作装备                   │
│  > [3] 与村长交谈                            │
│  > [4] 查看背包                             │
│                                           │
│  最近日志：                                  │
│  [10:23] 你获得了 5 木材                    │
│  [10:21] 体力已恢复                          │
│                                           │
└─────────────────────────────────────────────┘
```

### 设计原则
1. **极简主义** - 仅用文字和简单符号
2. **沉浸感** - 详细的文字描述，氛围营造
3. **响应式** - 支持键盘操作和点击
4. **即时反馈** - 每个操作都有文字反馈

## 🎨 视觉风格指南

### 配色方案（暗色主题）
```css
/* 背景 */
bg-primary: #1a1a2e;    /* 深蓝黑 */
bg-secondary: #16213e;  /* 次背景 */
bg-tertiary: #0f3460;   /* 高亮背景 */

/* 文字 */
text-primary: #e94560;  /* 主强调色（红色） */
text-normal: #ecf0f1;   /* 正常文字 */
text-dim: #95a5a6;      /* 次要文字 */

/* 状态 */
success: #27ae60;
warning: #f39c12;
danger: #e74c3c;
info: #3498db;
```

### 字体层级
```css
/* 标题 */
font-size: 1.5rem;
font-weight: bold;

/* 正文描述 */
font-size: 1rem;
line-height: 1.6;

/* 选择项 */
font-size: 1rem;
color: #3498db;  /* 蓝色链接感 */

/* 日志/状态 */
font-size: 0.875rem;
color: #7f8c8d;
```

## 🎮 交互命令设计

### 通用命令
| 命令 | 快捷键 | 说明 |
|------|--------|------|
| 1-9 | 数字键 | 选择对应选项 |
| I | i | 打开背包 |
| C | c | 角色状态 |
| S | s | 系统设置 |
| ESC | esc | 返回/取消 |

### 场景特定命令
**森林场景:**
- [1] 采集木材
- [2] 狩猎兔子
- [3] 采集浆果

**战斗场景:**
- [1] 攻击
- [2] 使用物品
- [3] 逃跑
