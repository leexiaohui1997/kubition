# M7: 探索系统

## 📋 里程碑目标

实现多地点探索，解锁更多采集区域。

**工期:** 2 天
**前置:** M6 地牢系统
**输出:** 多地点切换 Demo

---

## ✅ 核心功能

### 1. 新地点
- **溪流**: 水源、有毒植物
- **矿洞**: 铁矿、硝石
- **荒原**: 低温环境、高级猎物

### 2. 环境效果
- 温度影响角色状态
- 某些地点需要特定装备
- 旅行时间成本

### 3. 地图系统
显示已解锁地点

---

## 🎯 验收标准

- [ ] 解锁并访问3+新地点
- [ ] 新地点有不同资源
- [ ] 环境温度影响状态
- [ ] 地图上可快速导航
