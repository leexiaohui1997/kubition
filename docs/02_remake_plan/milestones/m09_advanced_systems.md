# M9: 进阶系统

## 📋 里程碑目标

实现农场、陷阱、科技树等进阶系统。

**工期:** 2 天
**前置:** M8 任务系统
**输出:** 农场 + 陷阱 + 科技系统

---

## ✅ 核心功能

### 1. 农场系统
- 田地块数升级
- 种植种子
- 时间成熟收获
- 种子：小麦、胡萝卜、甜菜

### 2. 陷阱系统
- 放置陷阱
- 定时捕获
- 捕获概率计算

### 3. 科技树
- 背包扩展
- 制造速度提升
- 采集效率提升

---

## 🎯 验收标准

- [ ] 种植并收获作物
- [ ] 陷阱捕获到动物
- [ ] 科技升级解锁新配方
