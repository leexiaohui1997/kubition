# M6: 地牢探险

## 📋 里程碑目标

实现地牢系统，玩家可以探索多层地牢并战斗。

**工期:** 3 天
**前置:** M5 战斗系统
**输出:** 可玩的地牢1-5层

---

## ✅ 核心功能

### 1. 地牢层级
- 每层有特定怪物池
- 精英怪和BOSS机制
- 层奖励结算

### 2. 特殊道具
- **穿洞绳**: 返回已到达层
- **火把**: 照亮周围，提高资源发现率
- **钥匙**: 打开宝箱

### 3. 探险选项
- [前进] 可能遭遇怪物
- [休息] 恢复但可能被偷袭
- [返回] 使用穿洞绳

---

## 🎯 验收标准

- [ ] 能进入地牢
- [ ] 每层有不同怪物
- [ ] 击败怪物获得经验/掉落
- [ ] 到达深层获得更好奖励
- [ ] 可以安全返回城镇
